About Me

Hi there! I'm Ümit Kara. I'm a computer engineer from in Sakarya, Turkey. I was born in 1997 in Istanbul. My enthusiasm about computers started when i was a little kid. I met programming at primary school. I started to learn Visual Basic. I build my first program with it, a calculator. After that my curiosity started to grow. After a while a started to learn C# and build more visually enhanced programs with WPF. Also at that times i met with web development. I started to learn HTML, CSS and JavaScript. I build my first website as a school project. All of these experiences made me to be an engineer. I work on web and mobile development. Also i made data science and machine learning projects. I'm curious about all fields of computer science and i want to develop my skills as far as i can.
